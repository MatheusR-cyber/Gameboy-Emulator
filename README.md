# 🎮 Game Boy Emulator - 5 Jogos Clássicos

Um emulador interativo de Game Boy com 5 jogos clássicos implementados em HTML5, CSS3 e JavaScript vanilla.

## 🕹️ Jogos Inclusos

- **🧱 Tetris** - Clássico puzzle de blocos
- **🐍 Snake** - Jogo da cobrinha
- **🧠 Memory** - Encontre os pares
- **🐦 Flappy Bird** - Desvie dos obstáculos
- **🎯 Breakout** - Quebre todos os blocos

## 🎮 Controles

### Teclado
- **Setas do Teclado (↑↓←→)** - Movimento/Navegação
- **Z** - Botão A (ação principal)
- **X** - Botão B
- **Enter** - Start
- **Backspace** - Select

### Mouse/Touch
- Clique/Toque nos botões do Game Boy

## 🚀 Como Usar

1. Clone o repositório:
```bash
git clone https://github.com/MatheusR-cyber/gameboy-emulator.git
cd gameboy-emulator
```

2. Abra o arquivo `index.html` em seu navegador

3. Selecione um jogo do painel lateral

4. Divirta-se! 🎉

## 📊 Pontuação

### Tetris
- Linha completa: +100 pontos

### Snake
- Comer comida: +10 pontos

### Memory
- Par encontrado: +50 pontos

### Flappy Bird
- Passar por obstáculo: +10 pontos

### Breakout
- Destruir bloco: +10 pontos

## 🐛 Solução de Problemas

### Jogo não responde aos controles?
- Clique na área da tela para garantir o foco
- Tente usar os botões do Game Boy em vez do teclado

### Canvas não renderiza?
- Verifique se o navegador suporta HTML5 Canvas
- Limpe o cache do navegador
